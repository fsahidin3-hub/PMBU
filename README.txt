Undangan Digital Pengajian Rutin (Template GitHub Pages)
========================================================

Proyek ini adalah template undangan digital statis yang didesain agar mirip dengan poster "Pengajian Rutin Malam Jumat" dari PMBU. Ideal untuk digunakan pada GitHub Pages.

--------------------------------------------------------
DETAIL ACARA
--------------------------------------------------------
- Acara: Pengajian Rutin Malam Jumat
- Penceramah: Ustadz Sunarya
- Lokasi: Masjid Al Barokah N.A
- Sifat: Terbuka Untuk Umum

--------------------------------------------------------
STRUKTUR FILE PROYEK
--------------------------------------------------------
- index.html: Struktur halaman dengan detail Pengajian terbaru.
- style.css: Gaya visual (warna hijau, font, tata letak dua kolom poster).
- script.js: Logika JavaScript untuk personalisasi nama tamu (TIDAK BERUBAH).
- README.TXT: Informasi proyek ini.

[FILE GAMBAR]
Pastikan semua file gambar yang diunggah berada dalam folder yang sama:
- ustadz-sunarya.png
- logo-pmbu.png
- whatsapp-qr.png
- (Jika ada file terpisah) kaligrafi-arab.png

--------------------------------------------------------
FITUR PERSONALISASI (script.js)
--------------------------------------------------------
Fitur personalisasi nama tamu tetap berfungsi dengan parameter 'kpd' di URL. Ini berguna jika Anda ingin menyapa tamu secara spesifik.

1.  URL Dasar Anda: https://[username].github.io/[nama-repo]/

2.  Contoh URL Personal:
    Tambahkan "?kpd=" diikuti dengan nama tamu di akhir URL Anda.
    - Untuk "Jamaah Al-Barokah": Gunakan ?kpd=Jamaah%20Al-Barokah
    - Untuk "Bapak RT 01": Gunakan ?kpd=Bapak%20RT%2001

--------------------------------------------------------
CATATAN PENTING
--------------------------------------------------------
* Pastikan semua gambar sudah terunggah ke GitHub.
* Periksa index.html untuk memastikan tag gambar menunjuk ke nama file yang benar.
