
README - Template Undangan Pengajian (Shalawat)

File di dalam paket ini:
- index.html    -> Halaman utama undangan
- style.css     -> Styling & layout
- script.js     -> Kontrol audio dan smooth scroll
- assets/       -> Tempat untuk meletakkan file shalawat.mp3

Catatan penting tentang musik (shalawat):
- Saat ini file 'assets/shalawat.mp3' TIDAK termasuk di paket karena lisensi dan ukuran file.
- Agar musik shalawat bisa diputar otomatis, unggah file mp3 shalawat (format .mp3) ke folder 'assets' pada repository GitHub Anda, lalu namai file tersebut 'shalawat.mp3'.
- Alternatif: upload file mp3 ke layanan hosting file (mis. Google Drive public link atau layanan hosting lain), lalu ubah path <source src="..."> pada index.html sesuai URL file mp3 tersebut.
- Perlu diingat: banyak browser memblokir autoplay suara. Pengunjung biasanya perlu klik tombol 'Play Shalawat' untuk memulai audio.

Cara pakai cepat:
1. Ekstrak file zip.
2. Edit file index.html jika perlu ubah tanggal/waktu/alamat/nomor WA.
3. Upload seluruh isi folder ke repository GitHub (misal repo: PMBU) menggunakan 'Upload files' di GitHub.
4. Aktifkan GitHub Pages (Settings → Pages → Branch: main → /root → Save).
5. Akses: https://<username>.github.io/<repository>/
